Селекторы:
    
    1. Основные
        - по тегу, названию;
        - по названию классу;  . 
        - по названия идентификатора;  #
        - * все эелементы;
        
    . - показатель класса;
    # - признак идентификатора;
    * - выделяет все элементы;
    
    2. Иерархические
    
        - Селектор по дочерним эелементам - > ;
        - Селектор по потомкам и дочерним - _пробел_;
        - Селектор по сестринским
            * первый сестринский + ;
            * все сестринские ~ ;
            
            
            
        div>p {
            все теги p которые находятся внутри дивов
        }
        
        
        Селектор по потомкам;
        div p{
            выберутся все теги p, которые внутри div и дочерние и потомки
        }
        
        .ex>p.ex2 {
            все дочерние p у которых класс ex2
        }
        
        
        Сестринские элементы:
            - первый сестринский;
            
            img+div{
                назначает стилизацию первому сестринскому div
            }
            
            img~div{
                назначает стилизацию всем сестринским элементов
            }
