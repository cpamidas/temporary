    form без атрибутов - по-умолчанию GET метод, запрос будет уходить на текущий адрес
    
    2 стандартных атрибута:
    action - на какой скрипт или адрес уходит запрос;
    method - каким методом отправлять;
    
    input - строчный
    
    Атрибуты input:
    type - тип;
    name - идентификационное имя данных
    value - значение, типа заполненное в поле формы;
    placeholder - подсказка;
    
    value в type="submit" - название кнопки
    
    
    
    Кнопка для джаваскрипта. Без действий.
    
    Тег button:
